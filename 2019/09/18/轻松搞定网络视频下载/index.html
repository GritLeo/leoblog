轻松搞定网络视频下载

平时浏览视频时看到一些比较好的资源时都想下载下来“据为己有”，以便以后需要使用时方便一些，无赖各大视频网站对自己站点的视频设有保护措施，一般下载手段很难下载到，视频解析网站困于版权压力也不得不放弃对于主流视频网站的解析，今天安利一个好用的下载工具——you-get，you-get是一个基于 python 3的命令行下载工具，使用you-get之前一定要提前安装好Python3和FFmpeg,下面是you-get的正确打开方式：

一、安装Python3
二、安装FFmpeg
三、安装you-get
四、使用you-get下载视频

一、安装Python3

1）进入Python官方网站下载Python3，根据安装提示安装好软件。

2）在CMD命令行窗口下输入：Python，如显示以下内容，则代表安装成功。

二、安装FFmpeg

1）这里以Windows平台为例，进入FFmpeg官网。在【get the packages】一栏将鼠标移入windows图标，点击【windows Builds】中的”Builds”，如图：

2）在新打开的页面中选择【Static】版本，点击【Download Build】开始下载，如图：

3）下载完成后得到一个名为”ffmpeg-20180908-4cfb01d-win64-static.zip”的压缩文件，将其解压至你想安装的位置，在解压后的文件夹里找到【bin】文件夹并进入此文件夹，复制其路径以便我们之后创建环境变量时要用。我这里的路径是：D:\Program Files\ffmpeg-20190918-53d31e9-win64-static\ffmpeg-20190918-53d31e9-win64-static\bin，如图：

4）添加【FFmpeg环境变量】。在“计算机”图标上单击鼠标右键，选择“属性”进入【系统】界面，在系统界面左边的菜单中选择“高级系统设置”，选择“高级”，点击“环境变量”。在【环境变量】面板找到系统变量【Path】,点击“编辑”，在【编辑环境变量】面板点击“新建”，然后将之前复制的路径粘贴进来，记住要在路径的末尾加上英文状态的分号【;】，依次点击【确定】即可。操作步骤如下图：

在CMD命令行窗口下输入：ffmpeg -version，如显示以下内容，则代表安装成功。

三、安装you-get

you-get是一个基于 python 3的命令行下载工具，可以使用Python的pip命令安装。在CMD命令行窗口下输入【pip install you-get】即可安装，输入【you-get -V】查看安装版本，如显示以下内容，则代表安装成功。
